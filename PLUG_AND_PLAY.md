# PLUG_AND_PLAY — solid-mechanics

> *Integration guide.*


## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library |
| `alloc` | yes | Allocator support |

## Integration

Import `solid_mechanics` in your project to access the functionality.

```
use solid_mechanics::{};
```

## Compatibility

- **Rust edition**: 2021+
- **Targets**: All tier-1 Rust targets
